# WebCore
CORSO NET CORE 

Progetto ASP.NET Core 2.0 utilizzato per il corso ASP.NET CORE. Sono presenti diversi esempi per
  - Configurazione applicazione
  - Actions e Controllers 
  - Views
  - Sessioni
  - Tag Helper
  - Cookie Authentication
  - IOptions
  - Razor syntax
  - IDataProtection
  - ByteArraModelBinder
  - HTTPS Redirect
  - MemoryCache
